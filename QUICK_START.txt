CoinFox - Quick Start Commands
================================

INSTALLATION:
npm install --legacy-peer-deps

START DEVELOPMENT:
npm start

BUILD FOR PRODUCTION:
npm run build

TEST APPLICATION:
npm test

DEPLOY TO GITHUB PAGES:
npm run deploy
